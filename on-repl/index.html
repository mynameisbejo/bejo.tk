First Impression - Install Htmly on Replit

Posted at — Mar 19, 2023

Semenjak tahu kelebihan dari replit yang bisa install hampir semua aplikasi untuk privacy seperti simply translate, libreddit, librex, searxNG dan banyak lainnya. Saya ingin mencoba ngeblog menggunakan HTMLy di replit.

Dan seperti yang anda lihat, blog ini menggunakan replit sebagai hostingnya dan HTMLy sebagai CMSnya.

Tapi, banyak sekali masalah yaitu deprecated dari versi PHP saat ini. Karena update terakhir HTMLy hampir setahun yang lalu yaitu 18 Mei 2022.

Dan karena replit versi gratis bisa dilihat oleh orang lain, password yang ada di file config.ini juga pasti terlihat jika ada yang tahu alamat asli blog direpl.

Karena itu saya tidak jadi menginstall HTMLy di repl dan mengubah CMSnya ke Hugo seperti biasa karena saya sudah terbiasa dengan Hugo dan ada blog lama yang beralamat di bebi.gotdns.ch (nama domain expired dan tidak bisa diklaim lagi).

Blognya memiliki satu posting yang bertanggal 12 Oktober 2019, cukup lama tersimpan di hardisk.

Akhirnya saya memutuskan untuk menggunakan template blog lama ini menjadi blog bejo.tk.

Oh iya, selama ini domain bejo.tk saya arakan ke soup.io, karena sudah tidak aktif lagi dan tidak relevan untuk saat ini, blog ini saya carikan hosting gantinya.

Walaupun sebenarnya lebih mudah dihosting menggunakan vercel, netlify, github pages, dll

Tapi kan lebih enak punya beberapa website dengan hosting berbeda.
